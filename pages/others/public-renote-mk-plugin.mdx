# Misskey Plugin "Public Renote"

デフォルトの公開範囲に関わらず、パブリックRenoteをするアクションを追加するプラグイン

```aiscript
/// @ 0.13.3
### {
  name: "Public Renote",
  version: "1.0",
  author: "unishigured",
  description: "デフォルト公開範囲に関わらず、パブリック設定でRenoteする",
  permissions: ["write:notes"],
  config: null
}

Plugin:register_note_action("Public Renote", @(note) {
    Mk:api("notes/create", { visibility: 'public', renoteId: note.id })
})
```
