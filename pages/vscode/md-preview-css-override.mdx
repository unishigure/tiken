# 拡張機能 Markdown Preview Github Styling の CSS 上書き

## Overview

VSCode 拡張機能 [Markdown Preview Github Styling](https://marketplace.visualstudio.com/items?itemName=bierner.markdown-preview-github-styles) は、ビルトインの Markdown プレビューを、Github 風に変更してくれるもの<br/>
この CSS を上書きし、更に好みの外観に変更する

## Content

1. ワークスペース設定で`markdown.styles`に CSS ファイルを設定

```json filename="settings.json"
{
  "markdown.styles": [".vscode/style.css"]
}
```

<Callout type="warning" emoji="⚠️">
  以下の仕様のため、共通設定とすることは出来ない模様
  > 実はワークスペース以下しか見ない仕様。
</Callout>

2. `github-markdown-body`要素に対して、変更したい CSS を記述

> Make sure you apply your styles to the .github-markdown-body instead of just body so they don't get overridden by this extension

```css filename="style.css"
.github-markdown-body {
    // favourite styles
}
```
